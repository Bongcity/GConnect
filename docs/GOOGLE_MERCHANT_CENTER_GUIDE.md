## 개요

### Google Shopping이란?

Google Shopping은 사용자가 Google 검색에서 직접 상품을 검색하고 비교할 수 있는 서비스입니다.

### GConnect의 역할

```
네이버 스마트스토어 → GConnect → Google Merchant Center → Google Shopping
```

GConnect는 네이버 상품을 Google이 이해할 수 있는 형식(XML 피드)으로 변환하여 자동으로 동기화합니다.

## Google Merchant Center 계정 생성

### 1단계: 계정 만들기

1. https://merchants.google.com 접속
2. Google 계정으로 로그인
3. **시작하기** 클릭
4. 국가 선택: **대한민국**
5. 비즈니스 이름 입력
6. 시간대 선택: **서울**

### 2단계: 비즈니스 정보 입력

```
비즈니스 이름: [네이버 스마트스토어 상호명]
웹사이트 URL: https://smartstore.naver.com/[your-store]
고객 서비스 이메일: [이메일]
고객 서비스 전화번호: [전화번호]
```

### 3단계: 웹사이트 인증

**방법 1: HTML 태그 (권장)**
1. Merchant Center에서 제공하는 메타 태그 복사
2. 네이버 스마트스토어 관리자 → 디자인 설정
3. HTML 헤더에 메타 태그 추가
4. Merchant Center에서 **인증** 클릭

**방법 2: HTML 파일 업로드**
1. 제공된 HTML 파일 다운로드
2. 네이버 스마트스토어 루트에 업로드
3. Merchant Center에서 **인증** 클릭

### 4단계: 비즈니스 정보 확인

- 사업자 등록증 업로드
- 사업자 등록번호 입력
- Google의 검토 대기 (보통 3~5 영업일)

## 상점 정보 설정

### Google Merchant Center 설정

#### 배송 설정

1. **도구 및 설정** → **배송 및 반품**
2. **배송 서비스 추가**
3. 배송 정보 입력:

```
배송 서비스 이름: 기본 배송
배송 지역: 대한민국 전체
배송비: 
  - 무료 배송: ₩30,000 이상 구매 시
  - 일반 배송: ₩3,000
배송 시간: 2~3일
```

#### 세금 설정

1. **도구 및 설정** → **세금**
2. **세금 규칙 추가**
3. 대한민국 세금 규칙 설정:

```
세율: 10% (부가가치세)
적용 상품: 모든 상품
```

#### 반품 정책

1. **도구 및 설정** → **반품 정책**
2. 정책 추가:

```
반품 기간: 7일
반품 배송비: 고객 부담
교환 가능 여부: 가능
```

## GConnect 피드 생성

### 1단계: 피드 설정 페이지 이동

1. GConnect Seller 로그인
2. **설정** → **Google 피드** 탭 클릭

### 2단계: 피드 정보 입력

```yaml
피드 제목: "내 스마트스토어 상품 피드"
피드 설명: "네이버 스마트스토어 상품 목록"
상점 URL: "https://smartstore.naver.com/your-store"
Google Merchant Center ID: [발급받은 ID] (선택)
```

### 3단계: 옵션 설정

- ✅ **자동 업데이트 활성화** (권장)
- **업데이트 주기**: 24시간마다 (권장)
- ☐ 비활성 상품도 포함 (선택)

### 4단계: 피드 생성

**피드 생성/수정** 버튼 클릭

생성된 피드 URL 예시:
```
http://localhost:3003/api/feed/[your-user-id]
```

⚠️ **중요**: 실제 운영 시에는 HTTPS 도메인 사용 필수!

## 피드 등록

### Google Merchant Center에 피드 등록

#### 1단계: 피드 메뉴 이동

1. Merchant Center 홈
2. **제품** → **피드** 클릭
3. **➕ 피드 추가** 클릭

#### 2단계: 국가 및 언어 선택

```
국가: 대한민국
언어: 한국어
대상: Shopping 광고 및 무료 목록
```

#### 3단계: 피드 방식 선택

**예약된 가져오기** 선택 (자동 동기화)

#### 4단계: 피드 정보 입력

```
피드 이름: GConnect 상품 피드
입력 방법: URL
URL: [GConnect에서 생성한 피드 URL 붙여넣기]
파일 이름: feed.xml
가져오기 빈도: 매일 (권장)
가져오기 시간: 02:00 (새벽 2시)
시간대: (GMT+09:00) 서울
```

#### 5단계: 피드 저장

**피드 만들기** 클릭

## 상품 승인 확인

### 피드 처리 상태 확인

1. **제품** → **피드** → 생성한 피드 클릭
2. **처리** 탭에서 상태 확인:
   - ✅ **성공**: 문제없이 처리됨
   - ⚠️ **경고**: 일부 항목에 문제 (선택사항 누락)
   - ❌ **오류**: 필수 항목 누락 또는 형식 오류

### 상품 목록 확인

1. **제품** → **모든 제품** 클릭
2. 동기화된 상품 확인:
   - **승인됨**: Google Shopping 노출 가능
   - **대기 중**: Google 검토 중
   - **거부됨**: 정책 위반 또는 오류

### 상품 거부 사유

일반적인 거부 사유:
- ❌ 이미지 품질 불량 (최소 100x100px)
- ❌ 정책 위반 상품 (가품, 위험물 등)
- ❌ 가격 불일치
- ❌ 상품 설명 부족

## 문제 해결

### 피드 오류 해결

#### "필수 속성 누락" 오류

**원인**: 상품에 필수 정보가 없음

**해결**:
1. GConnect Seller → 상품 관리
2. 해당 상품 편집
3. 누락된 정보 입력:
   - 상품명 (150자 이내)
   - 가격
   - 이미지 URL
   - 상품 URL

#### "이미지를 가져올 수 없음" 오류

**원인**: 이미지 URL 접근 불가

**해결**:
- 이미지 URL이 HTTPS인지 확인
- 이미지 크기: 최소 100x100px
- 파일 형식: JPG, PNG, GIF, BMP, TIFF

#### "가격 형식 오류" 오류

**원인**: 가격 형식이 잘못됨

**해결**:
- 가격은 숫자만 입력
- 소수점 사용 가능 (예: 19900.00)
- 통화는 자동으로 KRW 적용

### 피드가 업데이트되지 않음

**체크리스트**:
1. GConnect 설정 → Google 피드 → **자동 업데이트** 활성화 확인
2. Merchant Center → 피드 → **가져오기 빈도** 확인
3. GConnect 상품 관리 → **상품 동기화** 수동 실행
4. 피드 URL 직접 접속하여 XML 확인

### 상품이 승인되지 않음

**체크리스트**:
1. Merchant Center → 도구 및 설정 → **계정 문제** 확인
2. 상품별 거부 사유 확인
3. Google Shopping 정책 준수 확인:
   - https://support.google.com/merchants/answer/6149970

### 상품이 Google 검색에 안 나옴

**승인 ≠ 노출**

승인 후에도 노출까지 시간 소요:
- 신규 상품: 1~3일
- 기존 상품 수정: 몇 시간

노출 최적화:
- 고품질 이미지 사용
- 상세한 상품 설명
- 경쟁력 있는 가격
- 좋은 판매자 평가

## 성과 측정

### Google Merchant Center 인사이트

1. **실적** → **개요**
2. 확인 가능한 지표:
   - 노출수 (Impressions)
   - 클릭수 (Clicks)
   - 클릭률 (CTR)
   - 전환수 (Conversions)

### GConnect 분석 대시보드

1. GConnect Seller → **성과 분석**
2. 확인 가능한 데이터:
   - Google 검색 노출 수
   - Google 유입 클릭
   - 상품별 성과
   - 트래픽 소스
