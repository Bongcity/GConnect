# GConnect 가격 정책

## 📋 플랜 요약

| 플랜 | 가격 (월) | 상품 수 | API 호출 | 주요 기능 |
|------|-----------|---------|----------|-----------|
| **FREE** | 0원 | 5개 | 100회 | 무료 체험 |
| **Starter** | 39,000원 | 10개 | 1,000회 | 소규모 스토어 |
| **Pro** | 59,000원 | 50개 | 5,000회 | 성장하는 스토어 |

## 🎯 플랜별 상세 정보

### FREE (무료 체험)
- **가격**: 0원/월
- **상품 수**: 최대 5개
- **API 호출**: 100회/월
- **기능**:
  - 최대 5개 상품 동기화
  - 기본 SEO 구조화
  - 기본 통계
  - 자동 상품 동기화

---

### Starter
- **가격**: 39,000원/월 (468,000원/연)
- **상품 수**: 최대 10개
- **API 호출**: 1,000회/월
- **설명**: 소규모 스마트스토어를 위한 기본 플랜
- **기능**:
  - 최대 10개 상품 동기화
  - 기본 SEO 구조화
  - 기본 통계/리포트
  - 자동 상품 동기화
  - 이메일 지원

---

### Pro
- **가격**: 59,000원/월 (708,000원/연)
- **상품 수**: 최대 50개
- **API 호출**: 5,000회/월
- **설명**: 성장하는 스토어를 위한 프로 플랜
- **기능**:
  - 최대 50개 상품 동기화
  - 고급 SEO 구조화
  - 고급 통계 (키워드/경쟁사/CTR 분석)
  - 우선 기술 지원
  - 자동 상품 동기화
  - API 연동

---

### Enterprise
- **가격**: 협의
- **상품 수**: 50개 초과 (사실상 무제한)
- **API 호출**: 무제한
- **설명**: 대규모 스토어 및 에이전시를 위한 커스텀 플랜
- **기능**:
  - 커스텀 SEO 구조화
  - 커스텀 연동/리포트
  - 전담 매니저
  - SLA 보장
  - 우선 기술 지원
  - 웹훅 연동 (Slack, Discord 등)
  - 맞춤형 기능 개발

## 🔄 변경 이력

### 2025년 1월 개편
**변경 전**:
- Starter (10K): 300,000원/월, 10,000개
- Pro (50K): 800,000원/월, 50,000개

**변경 후**:
- Starter: 39,000원/월, 10개
- Pro: 59,000원/월, 50개
- FREE: 0원/월, 5개 (신규)

**변경 사유**:
- 더 많은 소규모 셀러가 접근할 수 있도록 가격 인하
- 실제 사용 패턴에 맞춘 상품 수 조정
- 무료 체험 플랜 도입으로 진입 장벽 완화

## 💾 데이터베이스 업데이트

### SQL 스크립트 실행

```bash
# 방법 1: sqlcmd
cd D:\GConnect
sqlcmd -S localhost -d GConnect -E -i scripts\update-pricing-policy.sql

# 방법 2: SQL Server Management Studio
# 1. scripts/update-pricing-policy.sql 파일 열기
# 2. F5 키로 실행
```

### 주의사항
- ⚠️ **SQL 스크립트를 실행하기 전에 데이터베이스 백업 권장**
- 기존 사용자의 구독은 그대로 유지됩니다
- 새로운 가격은 신규 구독 및 갱신 시 적용됩니다
- `displayName`이 "Pro (50K)" → "Pro"로 변경됩니다
