피그마에 바로 붙여넣고 디자인 시작할 수 있게 **상품 사이트용 5,000자 이내 브리프**로 줄여볼게.

---

## 1. 사이트 개요

* 도메인: `www.gconnect.kr`
* 역할:

  * 네이버 스마트스토어 상품을 모아 **구글/유저에게 잘 보이게 하는 상품 허브**
  * 유저: 구글 → GConnect 상품 상세 → 네이버 상품 상세로 이동
* 핵심 기능:

  * 상품 검색/카테고리/상점별 보기
  * **최근 본 상품** 노출
  * **키워드 추천 자동완성**이 있는 검색 UX

---

## 2. 공통 디자인 가이드

* 컬러 (IR와 통일):

  * 배경: 딥 네이비/퍼플 그라디언트
  * 카드: 어두운 톤 + 살짝 투명/그라스 느낌
  * 포인트: 네온 그린/민트 (CTA 버튼, 강조)
* 레이아웃:

  * 최대 폭 1140~1200px 중앙 정렬
  * 상단 고정 헤더 + 푸터
* 공통 요소:

  * 상단 헤더에 로고 + 메뉴 + 검색창
  * 하단 푸터: 회사정보/약관/개인정보/IR 링크

---

## 3. 네비게이션 & 정보 구조

### 헤더

* 좌측: GConnect 로고
* 중앙/우측 메뉴:

  * 상품 검색(검색창)
  * 카테고리
  * 입점 상점
  * 이용 안내(`/guide`)
  * 고객센터(`/support`)
* 우측 버튼:

  * 상점 로그인 → `seller.gconnect.kr`
  * 입점 문의 → `ir.gconnect.kr` 또는 신청 페이지

### 주요 URL

* `/` : 메인
* `/search` : 검색 결과
* `/c/[...slug]` : 카테고리 페이지
* `/p/[productId]-[slug]` : 상품 상세
* `/shops`, `/shops/[shopId]` : 입점 상점 관련
* `/guide`, `/support` : 안내/고객센터

## 4. 페이지별 디자인 가이드

### 4-1. 메인 `/`

**(1) 히어로**

* 좌측:

  * 타이틀:

    * “스마트스토어 상품, **구글에서 한 번에 찾기**”
  * 서브텍스트:

    * “네이버 스마트스토어 상품을 모아, 구글에 최적화된 상품 페이지로 보여줍니다.”
  * 큰 검색창:

    * placeholder: “상품명, 브랜드, 키워드로 검색해보세요”
    * 입력 시 자동완성 드롭다운(키워드 추천)
* 우측:

  * 카드형 영역: 인기 상품/카테고리 미리보기 (슬라이더 또는 그리드)

**(2) 인기 카테고리**

* 2줄 그리드 카드 6~8개 (패션/리빙/디지털/뷰티/식품 등)
* 카드: 아이콘/이미지 + 카테고리명 → 클릭 시 해당 카테고리 페이지

**(3) 입점 상점 추천 상품**

* 타이틀: “GConnect 입점 상점 추천 상품”
* 가로 스크롤 또는 4~6개 그리드
* 카드: 이미지/상품명/가격/상점명 + “입점” 뱃지

**(4) 지금 많이 보는 상품**

* 최근 클릭/조회 기준 Top 상품 리스트
* 카드 구조 동일

**(5) 최근 본 상품 (localStorage 기반)**

* 메인 하단 섹션: “최근 본 상품”
* 가로 슬라이더 카드 (썸네일/상품명/가격)
* localStorage에 데이터 없으면 섹션 숨김

**(6) 이용 안내 요약 + 푸터**

* 3단계 텍스트:

  1. GConnect에서 상품 검색
  2. 상세정보 확인
  3. 네이버에서 결제/배송
* `/guide` 링크, 푸터 공통 정보

---

### 4-2. 검색 결과 `/search`

* 상단:

  * 상단 고정 검색창(입력값 유지)
  * 키워드: “`무선 이어폰` 검색 결과 1,234개”
  * 정렬 드롭다운: 추천순/최신순/가격낮은순/가격높은순
* 좌측:

  * 필터 패널: 카테고리 트리, 가격 범위, 상점 필터,
    체크박스(입점 상점만, 품절 제외)
* 우측:

  * 상품 리스트 (2~3열 그리드 또는 리스트형)
  * 카드: 썸네일, 상품명, 가격, 상점명+등급, “네이버 스마트스토어 판매” 문구
    → 클릭 시 상품 상세로 이동

---

### 4-3. 카테고리 `/c/[...slug]`

* 상단:

  * breadcrumb (홈 > 대분류 > 중분류 > 소분류)
  * 카테고리 타이틀 + 2~3줄 설명
* 본문:

  * 좌측: 하위 카테고리 리스트(필터/탭)
  * 우측: 상품 리스트 (검색 결과와 동일 구조)

---

### 4-4. 상품 상세 `/p/[productId]-[slug]`

* 상단:

  * 좌측: 이미지 슬라이더
  * 우측:

    * 상품명(H1)
    * 가격
    * 주요 옵션(색상/용량 등)
    * 상점명(네이버 스마트스토어), 상점 등급, 입점 뱃지
    * 버튼:

      * 주요 CTA: **“네이버에서 구매하기”** (새 탭)
      * 서브: “이 상점의 다른 상품 보기”
* 중단 탭:

  * [상품 정보]: 스펙 표 (브랜드/모델/카테고리 등)
  * [상세 설명]: 연동된 설명(스크롤)
  * [관련 상품]: 동일 카테고리/브랜드 상품 카드 리스트
* 하단(선택):

  * “최근 본 상품” 작은 섹션(메인과 동일 데이터 활용)

---

### 4-5. 입점 상점 `/shops`, `/shops/[shopId]`

* `/shops`:

  * 타이틀: “입점 상점”
  * 상점 카드 리스트: 상점명/로고/간단 설명/대표 카테고리 + “상품 보러가기”
* `/shops/[shopId]`:

  * 상단: 상점 프로필(이름/등급/설명)
  * 하단: 해당 상점의 상품 리스트 (검색 페이지와 유사)

---

### 4-6. 이용 안내 `/guide`

* GConnect 사용 흐름 설명:

  * GConnect는 검색/연결만 제공, 구매/배송/환불은 네이버에서 처리
* 시각적 플로우 다이어그램 추천

### 4-7. 고객센터 `/support`

* FAQ 리스트
* 문의 방법: 이메일/링크 정도만 (UI는 단순 카드/리스트)

---

## 5. 특별 기능 UX 가이드

### 5-1. 최근 본 상품

* 저장: 브라우저 localStorage (`gconnect_recent_products`)
* 상품 상세 페이지 진입 시:

  * 상품 ID/이름/이미지/가격을 배열로 저장 (중복 제거, 최대 10개)
* 노출:

  * 메인 하단 “최근 본 상품” 섹션
  * (선택) 상품 상세 하단에도 동일 디자인 재사용

### 5-2. 키워드 추천 자동완성

* 검색창 입력 시 아래 드롭다운 패널:

  * 섹션 1: “최근 검색어” (localStorage 저장 값)
  * 섹션 2: “인기 검색어” (서버 제공 상위 검색어)
* 각 아이템 클릭 시:

  * 검색창에 채워지고 바로 검색 실행 또는 엔터 대기
* 오른쪽에 작은 “×”로 최근 검색어 삭제 가능
